# Dual Trade

Landing page profesional para Dual Trade, comunidad de trading
de criptomonedas basada en análisis técnico y fundamental.

## Stack

- Next.js 16 (App Router, Turbopack)
- TypeScript
- Framer Motion
- TradingView Widgets

## Secciones

- Hero
- Ticker de criptomonedas en tiempo real
- Nuestra motivación
- Metodología (Análisis Técnico + Fundamental con widgets)
- Nuestros resultados
- Contáctanos
